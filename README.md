# Cyclistic Bike-Share Analysis
  ## Google Data Analytics Certificate — Capstone Project

  ### Overview
  Analysis of 5.4 million bike trips (March 2025 – February 2026) to identify behavioural differences between casual riders and annual members of Cyclistic, a bike-share company in Chicago.

  **Business Question:** How do annual members and casual riders use Cyclistic bikes differently?

  ---

  ### Tools Used
  - **SQL (Google BigQuery)** — data cleaning and analysis
  - **Tableau Public** — data visualisation
  - **Google Sheets** — data exploration

  ---

  ### Key Findings
  - Members take **64.6%** of all rides; casual riders **35.4%**
  - Casual riders average **19.4 min** per ride vs **11.75 min** for members
  - Casual riders peak on **weekends**; members peak **Tuesday–Thursday**
  - Casual ridership drops **92%** in winter vs **75%** for members

  ---

  ### Top 3 Recommendations
  1. **Weekend & Summer Membership Campaign** — target casual riders during peak usage moments
  2. **Flexible Weekend Membership Tier** — lower-cost entry point for weekend-only riders
  3. **Ride Duration Cost Savings Campaign** — show casual riders the financial benefit of membership
